Problem Statement:

1. Building can have multiple floors and can have multiple elevators.
2. Elevators can be called from any floor and can travel to any floor.
3. Each elevator has a maximum capacity and weight limit.
4. The system schedules elevator movement based on requests and optimizes for efficiency
5. Users can request an elevator from any floor by pressing up/down buttons.
6. Inside the elevator, users can select destination floors.
7. The system prioritizes and schedules these requests efficiently. 
8. Elevators have indicators showing current floor and direction.
9. Emergency stop functionality must be available.
10. Overload detection prevents elevator movement if capacity is exceeded.
11. Door sensors prevent doors from closing if obstructed.
